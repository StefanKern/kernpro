## Usage in the Application

To preview and switch between templates:

1. **Navigate to Cover Letter Editor**

   - From Job Details page, click "Create Cover Letter"
   - Or navigate directly to `/cover-letter-editor`

2. **Select Template**

   - Find the "Template Style" dropdown in the header
   - Click to see all 7 options with descriptions
   - Select your preferred template

3. **Preview Updates Automatically**

   - PDF preview refreshes when template changes
   - Review the new design
   - Try different templates to compare

4. **Download**
   - Click "Download PDF" button
   - File saves as `cover-letter-{template-name}.pdf`

---

## Customization

Each template uses consistent content but applies different styling:

- **Fonts:** All use Helvetica (standard PDF font)
- **Sizes:** Name: 18-24pt, Body: 10.5-12pt
- **Spacing:** Optimized for each layout style
- **Colors:** Defined in `colorSchemes` object in service

To modify colors or spacing, edit the template methods in:
`apps/kernpro/src/app/services/pdf-generator.service.ts`
